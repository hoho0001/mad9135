- `Map` and `Set`
```javascript
let m = new Map();
m.set(key, value);
m.has(key);
m.get(key);
m.delete(key);
let arrK = m.keys();
let arrV = m.values();
m.clear();
let lenM = m.size;

let s = new Set();
s.add(value);
s.has(value);
s.delete(value);
s.clear();
let lenS = s.size;
```
